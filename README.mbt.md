A Game Boy emulator written in MoonBit

## Requirements
[SDL3](https://github.com/libsdl-org/SDL/blob/main/INSTALL.md)

## Usage

    $ moon run ./cmd/main/ --target native <rom_path>

| Key   | Button |
| :---: | :----: |
| `W`   | ↑      |
| `A`   | ←      |
| `S`   | ↓      |
| `D`   | →      |
| `J`   | A      |
| `K`   | B      |
| `U`   | Select |
| `I`   | Start  |
